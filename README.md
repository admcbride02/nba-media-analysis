# NBA Media Coverage Analysis

Analyzing shifts in sports media language around NBA stars pre and post the social media era (2009).

## Research Question
Has sports media coverage shifted from gameplay-focused language to money and off-court focused language in the social media era?

## Players
- Pre-2009: Michael Jordan, Kobe Bryant
- Post-2009: LeBron James, Stephen Curry

## Methods
Corpus linguistics, lexical field analysis, NLP

## Tools
Python, NewsAPI, ProQuest, pandas, spaCy
